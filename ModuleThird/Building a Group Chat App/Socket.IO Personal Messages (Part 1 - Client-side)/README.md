# Socket.IO Personal Messaging MERN App

This is a complete MERN stack sample project with personal messaging using Socket.IO.

## Features
- Node.js + Express backend
- MongoDB user and message storage
- Socket.IO real-time personal messaging
- React client with login, user list, and private chat

## Setup
1. Copy `.env.example` to `server/.env` and set your MongoDB URI.
2. From the project root:
   - `npm run install:all`
3. Start development servers:
   - `npm run dev`

## Notes
- Backend runs on `http://localhost:5000`
- React client runs on `http://localhost:5173`
- Use a MongoDB instance locally or Atlas for persistence.
