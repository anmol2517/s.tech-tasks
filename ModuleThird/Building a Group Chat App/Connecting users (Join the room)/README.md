# MERN Personal Chat Room

This project is a MERN stack chat application that allows two users to join the same unique room for private messaging. The room ID is generated by sorting the two user emails alphabetically, so the same pair always produces the same room.

## Features

- Unique room ID generation from two user emails
- Backend validation of recipient email before joining a room
- Socket.IO real-time chat room join and messaging
- MongoDB user storage with sample seed data
- React client UI for joining room and chatting

## Setup

1. Install dependencies:
   - `npm install --prefix server`
   - `npm install --prefix client`

2. Create `.env` in `server/` from `.env.example` and set your MongoDB URI.

3. Seed sample users:
   - `npm run seed --prefix server`

4. Start backend and frontend:
   - `npm start --prefix server`
   - `npm run dev --prefix client`

5. Open http://localhost:5173 in your browser.

## Room ID Rules

The room ID is generated by sorting both email addresses alphabetically and joining them with an underscore:

```js
const roomId = generateRoomId(userEmail, otherEmail);
```

This ensures:
- `alice@example.com` + `bob@example.com` => same room as
- `bob@example.com` + `alice@example.com`
