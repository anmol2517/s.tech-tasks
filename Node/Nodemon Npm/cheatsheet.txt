Node.js & Npm

--------------------------------------------------------

nodemon

- Role : Automatically restarts server on file changes.
- Install globally : npm install -g nodemon
- Use : Avoids manual server restart during development.

package.json

- Role : Stores project metadata, dependencies, and scripts.
-->> Key sections :

  >>  "dependencies" -> Packages needed to run the app.
  >>  "devDependencies" -> Packages needed only for development.
  >>  "scripts" -> Custom commands, e.g., "start": "node app.js".

Installing packages

-  Regular dependency : npm install <package-name> (adds to "dependencies")
-  Dev dependency : npm install <package-name> --save-dev (adds to "devDependencies")
-  Production install : npm install --production (installs only "dependencies")
-  Uninstall : npm uninstall <package-name>

package.json default fields

-  Created by npm init : name, version, main, scripts, author, license
-  Optional : repository, devDependencies, keywords
